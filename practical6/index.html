HTML Attributes

All HTML elements have attributes that will privde additional information about that particular element.
It takes 2 parameters, i.e., a name and a value

Points to remeber for attrubutes :

  • Attributes always come in name/value pairs like this : attribute_name = "value"
  • Attributes are always added to the start of an HTML element.
  • Attribute values should always be enclosed in quotes. Double style quotes (" ") are the most common,
    but single style quotes (' ') are also allowed.
