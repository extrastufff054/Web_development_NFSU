CSS Margins

With CSS, you have full control over the margins. These are properties for setting the margin for each side
of an element (top, right, bottom, and left).
CSS has properties for specifying the margin for each side of an element.

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
All the margin properties can have auto, length, %, and inherit properties.

Auto margin

You can set the margin property tro auto to horizontally center the element within its container. The element will then take up the specified width, and the remaining space will be equally between the left and right margins.

This div will be horizontally centered bcause it has margin : auto.
